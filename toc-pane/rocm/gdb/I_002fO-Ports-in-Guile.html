24.4.3.23 I/O Ports in Guile ¶

Scheme Procedure: input-port ¶

Return ROCgdb’s input port as a Guile port object.

Scheme Procedure: output-port ¶

Return ROCgdb’s output port as a Guile port object.

Scheme Procedure: error-port ¶

Return ROCgdb’s error port as a Guile port object.

Scheme Procedure: stdio-port? object ¶

Return #t if object is a ROCgdb stdio port. Otherwise return #f.
