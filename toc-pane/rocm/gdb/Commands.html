3 ROCgdb Commands ¶

You can abbreviate a ROCgdb command to the first few letters of the command name, if that abbreviation is unambiguous; and you can repeat certain ROCgdb commands by typing just RET. You can also use the TAB key to get ROCgdb to fill out the rest of a word in a command (or to show you the alternatives available, if there is more than one possibility).
