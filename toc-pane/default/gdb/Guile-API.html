24.4.3 Guile API ¶

You can get quick online help for ROCgdb’s Guile API by issuing the command help guile, or by issuing the command ,help from an interactive Guile session. Furthermore, most Guile procedures provided by ROCgdb have doc strings which can be obtained with ,describe procedure-name or ,d procedure-name from the Guile interactive prompt.
