24.4.3.2 Guile Configuration ¶

ROCgdb provides these Scheme functions to access various configuration parameters.

Scheme Procedure: data-directory ¶

Return a string containing ROCgdb’s data directory. This directory contains ROCgdb’s ancillary files.

Scheme Procedure: guile-data-directory ¶

Return a string containing ROCgdb’s Guile data directory. This directory contains the Guile modules provided by ROCgdb.

Scheme Procedure: gdb-version ¶

Return a string containing the ROCgdb version.

Scheme Procedure: host-config ¶

Return a string containing the host configuration. This is the string passed to --host when ROCgdb was configured.

Scheme Procedure: target-config ¶

Return a string containing the target configuration. This is the string passed to --target when ROCgdb was configured.
