7.44 .gnu_attribute tag,value ¶

For Object Attributes v1, record a GNU object attribute with the pair tag, value for this file.

For Object Attributes v2, record an object attribute with the pair tag, value in the current subsection for this file.

tag can either be an integer value, or a known named key. value can either be an integer or a string. For Object Attribute v2, the expected value type depends on the type set on the subsection.

See Object Attributes.
